2.) Aktualität von Junie überprüfen und aktualisieren
- In PyCharm: File > Settings > Plugins > Installed > Junie
- Auf der rechten Seite wird die aktuelle Version angezeigt
- Wenn ein Update verfügbar ist, erscheint ein "Update" Button
- Klicke auf "Update", um Junie auf die neueste Version zu aktualisieren
- Nach der Aktualisierung starte PyCharm neu, um die Änderungen zu übernehmen

3.) Aktualität des Datenbestandes von Junie ausgeben
- Führe das Script "Scripts\junie_data_status.py" aus, um den Status des Datenbestandes von Junie anzuzeigen
- Das Script zeigt folgende Informationen an:
  - Anzahl der Junie-Datendateien
  - Datum der letzten Aktualisierung
  - Alter der Daten (in Tagen und Stunden)
  - Status (AKTUELL oder VERALTET)
  - Empfehlungen zur Aktualisierung, falls die Daten veraltet sind
- Beispiel für die Ausführung:
  python Scripts\junie_data_status.py
